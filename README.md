# libmach: C API to Mach core and Mach engine

* `libmachcore`: C API to Mach core (Window + Input + Graphics, nothing else)
* `libmachengine`: C API to fully-featured Mach engine

## Experimental

This is an **experimental** project according to [our stability guarantees](https://machengine.org/about/stability):

> When a project has an experimental warning, it means all bets are off. You should carefully read the warning to understand why the project is experimental, and assume the worst.

**Tracking issue:** https://github.com/hexops/mach/issues/858
